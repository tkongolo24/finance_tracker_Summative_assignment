About Expense Tracker

Our Mission

Expense Tracker is designed to help you manage your finances effortlessly. Track your spending, categorize transactions, and monitor your budget in a modern, intuitive interface.

Features

  • Track transactions with categories and dates
  • Dynamic dashboard with totals and insights
  • Search and filter transactions easily
  • Budget monitoring with alerts
  • Dark and Light mode toggle for comfortable viewing
  • Mobile-friendly and responsive design

Technologies

Built using HTML, CSS, and JavaScript, with modern features like Chart.js for dynamic graphs, localStorage for persistence, and modular code structure.
